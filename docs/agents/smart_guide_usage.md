# 智能引导功能使用指南

## 概述

智能引导功能允许机器人通过自然语言对话，带领用户到达指定目的地，并自动处理复杂地形（如台阶、斜坡）。

## 运行配置

```bash
# 启动智能引导模式（需要真实机器人或仿真）
dimos run unitree-go2-smart-guide --robot-ip 192.168.123.161

# 或使用仿真模式测试
dimos --simulation run unitree-go2-smart-guide
```

## 使用示例

### 示例1: 基本引导 - 去超市门口

**用户**: "带我去超市门口"

**Agent处理流程**:
1. 解析目的地（"超市门口"）
2. 查询地图获取位置
3. 启动引导模式
4. 语音确认: "好的，我带您去超市门口。请跟我来。"
5. 开始导航并监控地形

**代码实现**:
```python
# Agent内部调用
result = guide_to("超市门口", "")
speak(f"开始引导: {result}")
```

### 示例2: 带偏好的引导 - 走楼梯上二楼

**用户**: "带我去二楼会议室，我要走楼梯"

**Agent处理流程**:
1. 记录用户偏好: "走楼梯"
2. 规划经楼梯的路线
3. 接近楼梯时切换到爬楼梯模式
4. 完成后恢复正常行走

**代码实现**:
```python
# 启动引导（带用户偏好）
guide_to("二楼会议室", "走楼梯")

# 当检测到楼梯时自动处理
handle_stairs_encountered("up")
speak("正在爬楼梯，请小心跟随")
```

### 示例3: 处理突发地形 - 遇到台阶

**用户**: "前面有台阶，你能上去吗？"

**Agent处理**:
```python
# 分析前方地形
terrain = analyze_terrain_ahead()
# 返回: "检测到向上的台阶，建议切换爬楼梯模式"

# 切换到爬楼梯模式
set_gait_mode("climb")
# 或: climb_stairs()

speak("我切换到爬楼梯模式了，可以安全通过台阶")
```

### 示例4: 暂停和恢复

**用户**: "暂停一下，我需要休息"

```python
pause_guide()
speak("已暂停引导。您可以在准备好后继续。")
```

**用户**: "继续走吧"

```python
resume_guide()
speak("继续前往目的地。请跟我来。")
```

### 示例5: GPS坐标引导

**用户**: "带我去这个坐标: 37.8059,-122.4290"

```python
# Agent识别为GPS坐标
guide_to("37.8059,-122.4290", "")

# 自动使用GPS导航
```

## 技能详解

### 1. guide_to(destination, user_preference)

**参数**:
- `destination`: 目的地描述（支持自然语言、地名、GPS坐标）
- `user_preference`: 用户偏好（可选）
  - `"走楼梯"` / `"坐电梯"`
  - `"最近路线"` / `"最平坦路线"`
  - `"避开人群"`

**返回值**: 引导状态和预估信息

**示例**:
```python
guide_to("最近的洗手间", "无障碍通道")
guide_to("停车场B区", "")
guide_to("37.7749,-122.4194", "最近路线")
```

### 2. analyze_terrain_ahead()

**功能**: 分析前方地形状况

**返回值**:
- `"前方路面平坦，可以正常通行"`
- `"检测到向上的台阶，建议切换爬楼梯模式"`
- `"检测到向下的斜坡，建议减速下行"`

### 3. handle_stairs_encountered(direction)

**参数**:
- `direction`: `"up"` 或 `"down"`

**功能**: 自动处理楼梯场景

### 4. set_gait_mode(mode)

**参数**:
- `mode`: `"normal"`, `"climb"`, `"descend"`, `"slow"`

**功能**: 手动设置步态模式

### 5. 引导控制技能

- `pause_guide()`: 暂停引导
- `resume_guide()`: 恢复引导
- `cancel_guide()`: 取消引导
- `report_guide_status()`: 查询当前状态

## 扩展开发

### 添加新的地形处理

在 `terrain_aware_navigation.py` 中添加:

```python
@skill
def handle_rough_terrain(self) -> str:
    """处理崎岖不平的地形"""
    self.set_gait_mode("slow")
    # 其他处理逻辑
    return "已切换到崎岖地形模式"
```

### 添加语音引导提示

在 `smart_guide.py` 的 `_guide_loop` 中添加:

```python
# 在特定位置给出方向提示
if approaching_intersection:
    self._speak("前方路口右转")

if long_straight:
    self._speak("直走约20米")
```

## 注意事项

1. **安全第一**: 始终优先警告潜在危险
2. **用户确认**: 重要决策点（如楼梯）等待用户确认
3. **优雅降级**: 如果地形分析失败，切换到慢速模式
4. **语音反馈**: 保持适当的语音更新频率，避免过多干扰

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 找不到目的地 | 地图数据不全 | 使用GPS坐标或更通用的地标 |
| 无法检测地形 | 点云数据缺失 | 检查LiDAR连接，使用视觉备用方案 |
| 爬楼梯失败 | 步态配置不当 | 调整 body_height 和 foot_raise_height |
| 导航中断 | 障碍物阻挡 | 自动重规划或请求用户协助 |
